# WORLD

STATUS: WORLD
SCOPE: UNIVERSAL

## Purpose

Defines world-side existence.

This layer controls:

- civilization structure
- architecture language
- environmental condition
- atmosphere baseline

This layer defines WHAT KIND OF WORLD exists.

This layer must remain independent from:

- camera interpretation
- rendering philosophy
- character identity

---
