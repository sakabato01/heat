# VISUAL

STATUS: VISUAL
SCOPE: UNIVERSAL

## Purpose

Defines final visual interpretation.

This layer controls:

- rendering language
- scene composition
- camera interpretation

This layer defines HOW the world is visually observed.

This layer must NOT define:

- character identity
- civilization structure
