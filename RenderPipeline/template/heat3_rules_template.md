# RULE STACK

STATUS: GLOBAL
SCOPE: UNIVERSAL

## Purpose

Defines ownership separation,
priority hierarchy,
and conflict resolution
for all HEAT systems.

This layer does NOT define content.

---

# OWNERSHIP

## Character Owns

- identity
- silhouette
- recognition
- behavior
- surface organization

Character must remain stable across:

- rendering variation
- scene variation
- camera variation

---

## World Owns

- civilization structure
- architecture language
- environmental atmosphere
- world condition

World must remain independent from:

- character identity
- cinematic interpretation

---

## Visual Owns

- rendering interpretation
- scene composition
- camera observation
- cinematic readability

Visual must NOT overwrite:

- character identity
- world structure

---

# GLOBAL SUPPRESSION

## [SuppressionRules]

avoid ownership overlap,
avoid silhouette collapse,
avoid decorative overload,
avoid unstable readability,
avoid uncontrolled cinematic chaos

---

# GLOBAL PRIORITY

## [Priority]

identity stability
> silhouette readability
> world readability
> rendering coherence
> cinematic spectacle
> decorative complexity

---

# CONFLICT RESOLUTION

## [ResolutionOrder]

If conflict occurs:

1. preserve identity
2. preserve silhouette
3. preserve readability
4. preserve world coherence
5. suppress decorative excess

---

# RUNTIME ASSEMBLY

## [AssemblyOrder]

1. Character
2. World
3. Visual
4. Rules enforcement
