# State Layer

STATUS: STATE
SCOPE: MODIFIER

## Purpose

Defines temporary existence conditions.

This layer modifies how the character currently exists,
without redefining immutable identity.

State layers are responsible for:

- temporary behavior
- temporary structure exposure
- emotional intensity
- operational condition
- civilization mode

State layers must remain replaceable.
